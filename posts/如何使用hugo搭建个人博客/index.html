如何使用hugo搭建个人博客

大家好，今天我要给大家讲解一下“如何使用hugo搭建个人博客”。

什么是hugo

  • hugo,读音：喝油狗 翻译：雨果
  • Go语言实现的一个博客生成器
  • 世界上最快的博客生成器

如何安装hugo

本文只介绍windows的安装方式 1. 到 hugo下载地址 下载hugo_xxx_Windows-64bit.zip(32位系统下载hugo_xxx_Windows-64bit.zip) 2. 解压，将hugo.exe粘贴到 D:\Softwear\hugo\ 路径下（Softwear和hugo文件夹需要自己新建） 3. 把 D:\Softwear\hugo\ 加到path (添加到Path步骤) 3.1 右击 此电脑 3.2 属性 3.3 高级系统设置 3.4 环境变量 3.5 有上下2个框，建议在下面的框的 “变量” 栏中找到Path,双击 3.6 点击新建 3.7 输入D:\Softwear\hugo\ 3.8 确定后关闭所有对话框 4. 重启终端，运行

   hugo version
   Hugo Static Site Generator vxxx windows/amd64 BuildDate: xxxx-xx-xxxxx:xx:xxx

说明安装成功

快速搭建网站

新建本地仓库(creator目录)

  1. 进入hugo官网点击页面下方的 Quick Start

  2. （本步骤看完再操作！！！）新建一个没有文件的安全的目录下运行cmder,复制step2:create a new site中的代码

    hugo new site quickstart

    并运行（注意！！！quickstart为仓库名，要将这个名字换成你的github名字.github.io-creator。举个我的例子，我的github名字位ddb324,我就要输入

    hugo new site ddb324.github.io-creator

  3. 然后在cmder中输入

    code .

    意思是：用vscode打开当前目录。

  4. 用vscode新建终端(此时vscode应显示在creator目录中),输入step3中的第一行代码并运行

    git init

    意思是：该命令将创建一个名为 .git 的子目录，这个子目录含有你初始化的 Git 仓库中所有的必须文件。

  5. 继续输入step3中的第二行

    git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

    下载完成后输入第三句

  6. 继续输入step4中

    hugo new posts/my-first-post.md

    意思是创建一个新的名为《my-first-post》的文章，建议将‘my-first-post’改为‘开博大吉’

  7. 在vccode中打开‘开博大吉。md’,写入你的内容。 完成后将

      draft: ture
  改为
  draft: false

    draft:ture的意思是草稿状态，在这状态下不会发布，改成draft:false才能发布。

  8. 继续在终端中运行step5的step5

      hugo server -D

    用ctrl+鼠标点击 http://localhost:1313/ 类似这样的地址。这个地址是本地的预览地址。

  9. 在vscode中用ctrl+p打开step6中写的config.toml配置文件

    baseURL = "https://example.org/"
languageCode = "en-us"
title = "My New Hugo Site"
theme = "ananke"

    改为

    baseURL = "(你的github pages显示的地址)"
languageCode = "zh-Hans"
title = "xxx的博客"
theme = "ananke"

  10. 在vscode新建一个终端,输入step7的hugo -D完成网站的搭建。注意：此时网站只是在本地，需要将它上传到github.

将网站上传到github(重要)

新建本地仓库(public目录)

  1. 在creator目录中新建一个 .gitignore 文件

  2. 用vscode打开 .gitignore 里面输入

    /public/

    目的是在creator目录中的git本地仓库中忽略public目录，因为等下在public目录里还需要新建一个git本地仓库。

  3. 进入public目录（此时应在creator目录）

    cd public
git init
git add .
git commit -v

    在commit提示中输入‘第一次部署’，然后关闭文件。

新建远程仓库(public目录)

  1. 在github上新建一个仓库，仓库名为‘你的github用户名.github.io’,别的不用选，直接点击‘create repository’创建仓库。

  2. 分别复制这两行代码并在vscode的public目录下运行，注意要选择SSH（你的和我的不同）

    git remote add origin git@github.com:DDB324/ddb324.github.io.git
git push -u origin master

    目的是将public上传到一个单独的仓库

  3. 刷新github页面，点击settings,移动到github pages栏，下面有个类似https://ddb324.github.io/的网址，这就是你的博客的网址。（注意：这个网址也是上文在<如何搭建网站>的第9步中写的baseURL的网址）

新建远程仓库(creator目录)

  1. 在github上新建一个仓库，仓库名为‘你的github用户名.github.io-creator’,别的不用选，直接点击‘create repository’创建仓库。

  2. 分别复制这两行代码并在vscode的creator目录下运行，注意要选择SSH（你的和我的不同）

    git add .
git commit -m "backup"
git remote add origin git@github.com:DDB324/ddb324.github.io-creator.git
git push -u origin master

    目的是将creator上传到一个单独的仓库

到此就全部完成了，大家可以自己去试着操作下！
